This folder stores all uploaded note files.

IMPORTANT:
- Make sure this folder has write permissions (chmod 777 on Linux/Mac)
- On Windows with XAMPP, this should work by default
- All uploaded files are stored with unique names to prevent conflicts
- Supported file types: PDF, DOC, DOCX, TXT, PPT, PPTX
- Maximum file size: 10MB

DO NOT DELETE THIS FOLDER!
